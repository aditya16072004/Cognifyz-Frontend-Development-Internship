Cognifyz Front-End Development Internship Project
Overview

This project was developed as part of the Cognifyz Technologies Front-End Development Internship Program.

The website demonstrates various front-end development concepts ranging from basic HTML structure to advanced CSS preprocessing using Sass.

Technologies Used
HTML5
CSS3
JavaScript (ES6)
Bootstrap 5
Sass (SCSS)
JSONPlaceholder API
Tasks Completed
Level 1: Beginner
Task 1: Basic HTML Page
Created a structured HTML webpage
Added headings, paragraphs, and images
Task 2: Styling with Inline CSS
Applied inline styling
Modified text color, size, and background colors
Level 2: Intermediate
Task 3: Responsive Design
Implemented media queries
Optimized layout for mobile and desktop devices
Task 4: Interactive Button
Added JavaScript functionality
Dynamically changes background color when clicked
Level 3: Advanced
Task 5: API Integration
Used Fetch API
Retrieved and displayed user data from JSONPlaceholder
Task 6: Form Styling and Validation
Designed a contact form
Implemented client-side validation
Level 4: Expert
Task 7: Component-Based Styling
Utilized Bootstrap framework
Implemented responsive navigation and UI components
Task 8: CSS Preprocessing with Sass
Created reusable variables
Used nested styles
Compiled SCSS into CSS
How to Run
Download the project files.
Open index.html in your browser.

For Sass:

sass styles.scss styles.css

or

sass --watch styles.scss:styles.css
